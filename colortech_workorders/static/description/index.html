ColorTech - Órdenes de Trabajo

Gestión completa del taller de personalización

Módulo personalizado para ColorTech Guadalajara S.L., empresa dedicada a la pintura y personalización de dispositivos electrónicos.

Funcionalidades principales

  • Órdenes de Trabajo con flujo de estados: Recibido → En Proceso → Secado → Control de Calidad → Terminado → Entregado
  • Catálogo de Servicios con precios y tiempos estimados
  • Tipos de Dispositivo configurables (Móvil, Consola, Portátil, etc.)
  • Líneas de servicio con cálculo automático de costes
  • Historial de cambios en el chatter (mail.thread)
  • Datos de demostración incluidos para testing
  • Seguridad con permisos para usuarios y administradores

Modelos

  • colortech.workorder — Orden de Trabajo (modelo principal, 17+ campos)
  • colortech.workorder.line — Línea de Servicio (modelo secundario)
  • colortech.service — Catálogo de Servicios
  • colortech.device.type — Tipo de Dispositivo
